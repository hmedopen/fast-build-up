# Fast Build Up

Fast Build Up makes pillaring quick and smooth. Hold Jump and Use Item with a
placeable block in your main hand to place blocks beneath yourself while rising.

## Features

- Fast vertical building with normal block placement.
- Press `H` to enable or disable the mod.
- Sneaking disables placement by default.
- Configurable placement cooldown, vertical speed, water behavior, and centering.

## Requirements

Fast Build Up 1.1 targets Minecraft 26.1.2 with Fabric Loader 0.19.3,
Fabric API 0.151.0+26.1.2, and Java 25.

## Configuration

The first launch creates `config/fastbuildup.properties`. Edit it while the game
is closed to customize the placement and movement behavior.

## Building

```powershell
.\gradlew.bat build
```

The release JAR is written to `build/libs/fastbuildup-1.1.jar`.
